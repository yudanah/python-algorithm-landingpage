02 — AI Support

AI 튜터 보조

  • 막히는 순간 AI가 단계별 힌트 제공 (답을 바로 알려주지 않음)
  • 학생 수준에 맞춰 초보에겐 쉬운 말로, 고급자에겐 깊게 리뷰
  • 에러 메시지를 학생이 이해할 수 있는 말로 자동 번역
  • 수업 중 질문 폭주 없이 AI가 개념을 1:1로 설명
  • 정답과 학생 풀이 방식을 비교해서 더 나은 접근법 안내
  • AI 튜터 "나나"와 프로그래밍 주제 안에서 자유롭게 채팅
  • 최근 풀이 패턴을 분석해서 다음 학습 경로 자동 추천
  • 수업 시간에는 끄고, 자습 때는 켜는 학생별 AI 사용 ON/OFF

03 — Community

학생 참여형 커뮤니티

  • 학생이 직접 문제를 출제해서 "만드는 수업"으로 전환
  • AI가 출제된 문제를 자동 검수하고 난이도와 키워드까지 배정
  • 다른 학생이 내가 만든 문제를 풀어주는 구조로 출제자에게 최고의 복습
  • 좋아요, 즐겨찾기, 풀이 수 집계로 인기 문제가 자연스럽게 부각
  • 출제한 문제를 언제든 자유롭게 공개/비공개 전환 가능
  • 난이도별 검색, 키워드 필터로 원하는 문제를 빠르게 탐색
  • "이번 주 배운 내용으로 문제 1개씩 만들어오기" 같은 수업 활용에 적합

04 — Operations

기관 운영 관리

  • 반 생성, 학생 배치, 교사 배정을 한 화면에서 처리
  • 누가 몇 문제 풀었는지, 언제 접속했는지 실시간 확인
  • 풀이 기록, 코드 이력, 일별 차트, AI 내역까지 학생 상세 리포트
  • 학생별 개별 또는 반 전체 일괄 커리큘럼 설정 가능
  • 티켓으로 학생별 이용권 연장하고 잔여일을 실시간으로 확인
  • AI 학습 피드백을 링크 하나로 학부모에게 전달 (로그인 불필요)
  • 누가 언제 무엇을 변경했는지 활동 로그에 자동 기록
  • 학생 비밀번호 분실 시 선생님이 즉시 리셋 가능

Why Teams Choose It

도입하면 바로 달라지는 것들

01

수업 준비 시간이 절반으로 줄어요.

반별 진행 현황이 실시간으로 보이니까, 누가 어디서 막혔는지 수업 전에 미리 파악됩니다.

02

반복 질문은 AI가 대신 답해요.

"선생님 이거 어떻게 해요?"를 AI 힌트가 흡수합니다. 선생님은 개별 지도에 집중하세요.

03

학부모 상담이 데이터로 달라져요.

AI 학습 피드백 링크를 보내면 학부모가 로그인 없이 바로 확인합니다. 상담 신뢰도가 달라집니다.

04

학생이 스스로 들어와서 풀어요.

커뮤니티 출제, 복습 추적, 코스 추천이 학생의 자발적 학습 습관을 만들어줍니다.

05

모든 관리가 한 화면에서 끝나요.

클래스 생성, 교사 배정, 학생 관리를 별도 도구 없이 하나의 대시보드에서 처리합니다.
