# 랜딩페이지 기획서

> **작성일**: 2026-02-22
> **대상 도메인**: letscoding.co.kr (랜딩페이지)
> **서비스 도메인**: python.letscoding.co.kr (기존 학습 플랫폼)

---

## 1. 개요

### 1.1 목적

letscoding.co.kr에 접속하는 신규 방문자에게 파이썬 알고리즘 학습 플랫폼의 핵심 가치를 전달하고, 학원/학교 도입 문의로 전환시키는 랜딩페이지.

### 1.2 핵심 타깃

| 우선순위 | 타깃 | 니즈 |
|---------|------|------|
| 1순위 | 학원 원장 / 코딩 교육 담당자 | 체계적인 코딩 교육 커리큘럼, 학생 관리 도구 |
| 2순위 | 학교 교사 (방과후/정규) | 수업에 바로 쓸 수 있는 학습 도구, 진도 관리 |
| 3순위 | 학부모 | 아이의 코딩 학습 현황 확인, 신뢰할 수 있는 플랫폼 |
| 4순위 | 자기주도 학습 학생 (중고등) | 혼자서도 단계별로 배울 수 있는 환경 |

### 1.3 전환 목표 (CTA)

- **Primary**: "도입 문의하기" (기관용 상담 신청 폼)
- **Secondary**: "데모 영상 보기" (플랫폼 시연 영상)
- **Tertiary**: "python.letscoding.co.kr로 이동" (기존 사용자)

## 2. 기존 사용자 안내 팝업 레이어

### 2.1 사양

```
┌──────────────────────────────────────────────────────────────────┐
│  ℹ️  이미 사용 중이신가요?  [python.letscoding.co.kr 바로가기 →]  [✕]  │
└──────────────────────────────────────────────────────────────────┘
```

| 항목 | 내용 |
|------|------|
| 위치 | 페이지 최상단 고정 (sticky top bar) |
| 배경색 | 브랜드 포인트 컬러 (예: #2563EB 계열 블루) |
| 텍스트 | 흰색, 14~15px |
| 문구 | "이미 사용 중이신가요? **python.letscoding.co.kr** 바로가기 →" |
| 링크 | https://python.letscoding.co.kr (새 탭 열기) |
| 닫기 버튼 | 우측 [✕] 클릭 시 닫힘 |
| 닫기 유지 | `localStorage`에 저장하여 동일 브라우저에서 재방문 시 다시 표시하지 않음 |
| 반응형 | 모바일에서도 한 줄로 유지, 텍스트 축약 가능 ("학습하러 가기 →") |

### 2.2 동작 흐름

```
방문자 접속
  ├─ localStorage에 'popup_dismissed' 없음 → 팝업 표시
  │    ├─ "바로가기" 클릭 → python.letscoding.co.kr 새 탭 이동
  │    └─ [✕] 클릭 → 팝업 닫기 + localStorage 저장
  └─ localStorage에 'popup_dismissed' 있음 → 팝업 미표시
```

                                                              │
                                      ⑤ 정답!                │

## 4. 도입 문의 폼 (Contact Form)

"도입 문의하기" CTA 클릭 시 노출되는 모달 또는 별도 섹션.

### 4.1 입력 필드

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 기관명 | text | O | 학원/학교 이름 |
| 담당자 이름 | text | O | |
| 연락처 | tel | O | 전화번호 |
