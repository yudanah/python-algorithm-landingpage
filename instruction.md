## 1. 프로젝트 개요

### 제품명
**Let's Coding & Play** (렛츠코딩앤플레이)

### 한 줄 설명
학원을 위한 AI 기반 파이썬 코딩 교육 플랫폼

### 상세 설명
학생들이 게이미피케이션된 환경에서 파이썬 알고리즘을 학습하고, 선생님은 실시간으로 진행 상황을 모니터링할 수 있는 올인원 교육 솔루션입니다. AI 튜터가 개인 맞춤형 힌트와 피드백을 제공하여 학습 효율을 극대화합니다.

### 핵심 기능 요약

| 기능 | 설명 |
|------|------|
| 온라인 코드 에디터 | 브라우저에서 바로 Python 코드 작성 및 실행 |
| 자동 채점 시스템 | 테스트케이스 기반 즉각적인 정답 확인 |
| AI 학습 지원 | 힌트, 코드 리뷰, 에러 분석, 개념 설명 |
| 별 시스템 | 학습 성취도 시각화 (은색별/금색별) |
| 진행도 픽셀아트 | 500문제 달성까지 픽셀아트 완성 |
| 관리 대시보드 | 기관/클래스/학생 통합 관리 |
| 멀티테넌시 | 여러 기관/학원 독립적 운영 |

## 2. 타겟 고객

### Primary: 코딩 학원 원장/운영자
- **페인 포인트**: 학생 관리 어려움, 수업 품질 표준화, 강사 의존도
- **니즈**: 효율적인 학생 관리, 자동화된 채점, 학습 현황 파악

### Secondary: 코딩 강사/선생님
- **페인 포인트**: 수업 준비 시간, 개별 피드백 한계, 진도 관리
- **니즈**: 커리큘럼 제공, AI 보조, 실시간 모니터링

### Tertiary: 학부모
- **페인 포인트**: 자녀 학습 현황 파악 어려움
- **니즈**: 투명한 진행 상황, 신뢰할 수 있는 교육 품질

## 3. 핵심 가치 제안 (Value Proposition)

### 메인 카피
> **"AI와 함께하는 스마트한 코딩 교육"**
>
> 학생은 재미있게, 선생님은 편리하게, 학원은 효율적으로

### 서브 카피
> 자동 채점 × AI 튜터 × 실시간 관리 = 코딩 교육의 새로운 기준

### 3가지 핵심 USP

1. **AI 튜터**: 24시간 개인 맞춤형 학습 지원
2. **게이미피케이션**: 별 시스템과 픽셀아트로 학습 동기 유발
3. **올인원 관리**: 기관-클래스-학생 통합 대시보드

### 5.5 How It Works (작동 방식)

**헤드라인**: "3단계로 시작하세요"
**서브라인**: "복잡한 설정 없이, 가입 즉시 사용 가능"

**타임라인 레이아웃**:

```
    ① ─────────────── ② ─────────────── ③

[회원가입]         [클래스 생성]       [학습 시작]
학원 정보로        학생 계정 생성      학생은 문제 풀이
계정 생성          클래스 배정         선생님은 모니터링
```

**단계별 상세**:

| 단계 | 제목 | 설명 | 소요 시간 |
|------|------|------|----------|
| 1 | 회원가입 | 학원 정보를 입력하고 관리자 계정을 생성합니다 | 2분 |
| 2 | 클래스 생성 | 학생 계정을 만들고 클래스에 배정합니다 | 5분 |
| 3 | 학습 시작 | 학생은 문제를 풀고, 선생님은 실시간으로 확인합니다 | 즉시 |
